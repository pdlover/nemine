## About

Minimal minecraft server software, written in nelua

### Built With

* Nelua

* Minicoro

* Minilua

* Minifs

* Znet

## Getting Started

### Prerequisites

* Nelua and a C compiler

### Installation

> Warning: May not work on windows!

Just run `nelua -bo nemine nemine.nelua`

## Usage

Just run it in a console/terminal
